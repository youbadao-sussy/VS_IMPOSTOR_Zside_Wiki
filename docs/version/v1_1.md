# Version 1.1.0 - The Meltdown Update!
このアップデートには、新しいコンテンツが含まれる予定です。

## 追加要素
### 新しいリミックス曲
* Meltdown! - remix: 山猫 照

### 全般
* ゲームプレイ中の全イベントをかき直す。
* キャラクター設定
  * キャラクターが歌ったときに、カメラが動くようになる。
    * これは、キャラクターのjsonデータ内の`movement`の数値を変更して動きの大きさを変えたり、
      `Settings`->`Alt Gameplay`->`Camera Move on Note`のオンオフによって動作が変わります。
  * ノーツの同時押しでキャラクターのエフェクトを表示するようになる。
    * これは、キャラクターのjsonデータ内の`ghostEffect`のオンオフや、
      `Settings`->`Alt Gameplay`->`Double Ghost Effect`のオンオフによって動作が変わります。
  * キャラクター名、種族(crewまたはimpostor)のオプションを追加

### HUD関連
* HUDを自由に追加できるようになる。
* 体力バーIconを追加。
    * これは、現在制作中のHorizon Engineを参考にした新しいシステムによって作られており、Youbaはこれを`Icon Utility`と名付けました。
* SustainNoteは、一つの画像を盾に伸ばしたものとなる。

### イベント
* `Extra Cam Zoom`を追加。
* `Extra Cam Modulo`を追加。
* `Extra Cam Move`を追加。

### Freeplay
* Freeplayをリワーク。
